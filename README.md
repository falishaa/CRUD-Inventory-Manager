# Inventory Manager CRUD App
## Supra Coders Project by Falisha Quayum

**Inventory Manager** is a full-stack CRUD application that allows authenticated users to create, view, update, and delete inventory items. I built this for a Space Force program called Supra Coders.

Technologies Used:
* React
* Express
* SQLite
* JWT Authentication

App Features:
* User registration and login
* Create items
* Update items
* Delete items
* Public item browsing

How to Run:
1. Run backend with following commands:
```
npm install  
npm run dev
```

2. Open another terminal and run frontend with following commands:
```
npm install  
npm run dev
```

3. Navigate to the URL provided in console, it should look something like this:
```
http://localhost:5173/
```
